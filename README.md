## 💡 使用建议

### 1. 系统学习
按照「学习路径」从基础到高级逐步学习，建议每天学习 2-3 小时，持续 4-6 周。

### 2. 查漏补缺
根据面试需求，针对性查看对应分类：
- **准备基础面试** → 重点看 `01-JavaScript基础/` 和 `04-手写题/常用手写题_汇总.md`
- **准备框架面试** → 重点看 `02-框架原理/` 全部文件
- **准备算法面试** → 重点看 `03-算法与数据结构/算法.md`

### 3. 实战练习
- 结合手写题文档，动手实现代码
- 算法题建议在 [LeetCode](https://leetcode.cn/) 等平台同步练习
- 建立自己的代码仓库，保存练习代码

### 4. 原理深入
- 重点学习框架原理和工程化相关文档
- 结合源码阅读效果更佳
- 尝试手写简化版实现

### 5. 定期复习
- 建立学习笔记，定期回顾重点内容
- 制作思维导图，梳理知识体系
- 定期刷题，保持手感

## 📌 注意事项

### ✅ 学习建议
- 本知识库内容较多，建议分阶段学习
- 手写题需要实际编码练习，不能只看不练
- 算法题建议在 LeetCode 等平台同步练习
- 框架原理需要结合源码阅读效果更佳
- 建议配合项目实战，理论结合实践

### ⚠️ 学习误区
- ❌ 只看不练：手写题和算法题必须动手实现
- ❌ 死记硬背：理解原理比记忆更重要
- ❌ 贪多嚼不烂：建议按学习路径循序渐进
- ❌ 忽视基础：基础不牢，地动山摇
